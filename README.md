# React Booking System

A modern booking management system built with React, TypeScript, and Redux Toolkit.

## Project Summary

- Built a booking management interface with React and Redux Toolkit.
- Implemented full CRUD functionality for managing bookings.
- Added Cypress end-to-end tests to validate booking flows.

## Tech Stack

### Core

- React
- TypeScript
- Redux Toolkit

### Testing

- Cypress
- Jest

### Tooling

- ESLint
- Prettier

## Architecture Highlights

- Centralized state management using Redux Toolkit slices.
- Clear separation between UI components and state logic.
- End-to-end tests simulate real user booking flows.

## Key Features

- Create bookings
- Edit existing bookings
- Delete bookings
- Responsive interface
- Form validation and error handling

## Quality

- Unit testing with Jest
- End-to-end testing with Cypress
- Type-safe codebase with TypeScript
